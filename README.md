# cocojson

Utility scripts for COCO json annotation format

## Install

`cocojson` package can be installed through `pip3 install -e .` (editable install) or `pip3 install .`

## Usage

### Utility Tools

#### Merge

```python
python3 -m cocojson.merge_coco
```

#### Sample

```python
python3 -m cocojson.sample_coco
```
